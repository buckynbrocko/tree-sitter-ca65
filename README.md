# tree-sitter-ca65
A tree-sitter grammar for the ca65 assembler of the cc65 development package.

# Development

## Prerequisites

- [Node.js](https://nodejs.org/)
- [make](https://www.gnu.org/software/make/)
- [Emscripten](https://emscripten.org/)

## Quick Start

```shell
make init
```

## Pre-Commit

```shell
make pre-commit
```
